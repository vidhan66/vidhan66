## About Me

I'm an AI/ML Engineer with a deep interest in building intelligent systems that hold up under real world constraints. My work spans LLM-powered agents, RAG pipelines, and recommendation systems, and I think as much about failure modes, non-determinism, and scalability as I do about the models themselves.

Right now I'm going deeper into the infrastructure that makes AI fast and efficient — exploring GPU kernels, attention mechanisms like Flash Attention, and inference engines. The gap between a model that works and one that runs well at scale is where I want to operate.

I believe great ML engineering is 20% model and 80% knowing what to do when it fails.
